NixOS install (nixos-anywhere)

Run from your local machine. Target must be booted from a NixOS ISO with SSH reachable.

1. Scaffold a new host config

python3 ~/nix/scripts/new-host.py <hostname>

2. Install on the target

nix run github:nix-community/nixos-anywhere -- \ --flake ~/nix#<hostname> \ --generate-hardware-config nixos-generate-config \ root@<target-ip>

3. After first boot

scp root@<ip>:/etc/nixos/hardware-configuration.nix ~/nix/hosts/<hostname>/

Home-manager (non-NixOS)

curl -fsSL https://sejder.github.io/install_scripts/nix.py | python3
